# 🛠️ Setup 🛠️

Klona ner repot localt, installera och kör igång

```
git clone https://github.com/niklas-mardby/banana-ufo.git
cd banana-ufo
yarn
yarn build
yarn start
```

Med npm

```
git clone https://github.com/niklas-mardby/banana-ufo.git
cd banana-ufo
npm install
npm run build
npm run start
```
